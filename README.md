Сервис "Список фильмов"
Проект представляет собой веб-платформу для организации персональной видеотеки.

Пользователи получают возможность вести каталог просмотренных фильмов, добавлять новые киноленты, обновлять информацию о существующих записях и удалять устаревшие данные.

В системе реализованы функции сортировки и отбора контента по категориям и пользовательским оценкам.

Техническая реализация основана на современных фреймворках с обеспечением кросс-платформенной адаптивности интерфейса.

Основная задача проекта - отработка принципов гибкой методологии Scrum в условиях учебной разработки.
