Сервис "КиноГид"
Веб-платформа для каталогизации фильмов с функциями добавления, редактирования и удаления записей. Поддерживает фильтрацию по жанрам и оценкам. Реализована на современных технологиях с адаптивным дизайном. Цель — практика Scrum.
